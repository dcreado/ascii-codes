AsciiCodes
==========


Atom package to auto convert special characters into ASCII codes


**Supported files**

`*.html`, `*.htm`, `*.js`, `*.coffee`, `*.json`, `*.cson`, `*.php`, `*.java`, `*.jsp`, `*.xml`, `*.sql`


**How to use**

Enter any special character then press TAB


`å -> &aring;`

`á -> &aacute;`

`Ç -> &Ccedil;`


**Custom codes**

`ascii-heart -> &#9829;`
