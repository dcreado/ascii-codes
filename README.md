AsciiCodes
==========

_v0.1.1_


Atom package to auto convert special characters into ASCII codes


**Supported files**

`*.html`, `*.htm`, `*.js`, `*.coffee`, `*.json`, `*.php`, `*.java`, `*.jsp`, `*.xml`, `*.sql`


**How to use**

Enter any special character then press TAB


`å -> &aring;`

`á -> &aacute;`

`Ç -> &Ccedil;`
